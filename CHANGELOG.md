# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-23

### Added

- MCP integration for Windows-MCP server via `uvx windows-mcp`
- **Windows MCP Overview** skill — tool reference, selection guide, and decision tree for all 16 tools
- **Screen Interaction** skill — Click, Type, Scroll, Move, Shortcut, Wait patterns and best practices
- **Visual Capture** skill — Screenshot vs Snapshot guidance, DOM extraction, screen analysis strategies
- **App & Window Management** skill — launch, tile, switch, resize, and close application windows
- **System Operations** skill — PowerShell, process management, Windows Registry, and toast notifications
- **Web & Data Operations** skill — web scraping, clipboard automation, MultiSelect, MultiEdit patterns
- **Windows Automation Best Practices** skill — performance optimization, error handling, security, workflow design
- Reference files for each skill with advanced patterns, PowerShell one-liners, and troubleshooting guides
